# Prompt 07 — Notifications (Should Have)

## Prompt à copier-coller

Implémente un système de notifications in-app pour :
- Nouvelle réponse à une demande d'aide ou à un sujet de forum.
- Nouveau match mentor/étudiant proposé.
- Nouveau message privé reçu.

Fonctionnalités :
- Icône de notifications avec compteur non-lu dans le header.
- Liste des notifications avec lien direct vers l'élément concerné.
- Marquage comme lu.

Contraintes : Supabase Realtime ou triggers SQL + table notifications. Rester léger (pas de push notifications natives, hors périmètre MVP). Propose le plan avant d'exécuter.
